1. Introduction to the Web and HTML

1.1 How the Internet works

A web page is a text document written in a language called HTML (Hyper text markup language). Web browsers read these documents, and then interpret and display them.
Note: HTML = plain text + markup (the way it looks)
Most web pages consist of HTML, but but they can also contain plain text, images, videos, music or links to other pages.

1.2 Coding

Writing code means translating your thoughts into a language the computer can understand. However, computers are really stupid and will only do exactly what you tell them to, so be careful to get every detail right.
